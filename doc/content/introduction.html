简介

使用HTML的方式进行空间开发

EMR 全称 Easy Mixed Reality 其中 Easy 指的是快速、敏捷、使用简便，EMR使用基于空间视图的模式带你进入元宇宙的开发，如果您平时会上网、会写一个静态网页那么进入元宇宙的开发对你来说不难。

视图代码： 

<>
    <!--浩瀚的宇宙有颗美丽的星球住着有钱的人类-->
    <></SpaceNode>

    <>
        <!--地球的附近有个小弟它一直绕着地球转-->
        <></SpaceNode>

        <!--画个简陋的轨道月球不能出轨-->
        <></PanelRoot>
    </SpaceNode>
</Root>

组件逻辑： 

// mounted生命周期，这里空间中的内容已完全生成
protected override void mounted()
{
    // 通过Id获取空间内的指定节点
    var moonContainer = this.getNodeById<SpaceNode>("moonContainer");

    // 月球旋转，这里只是个示例为了演示说明，所以先让他转个 9999999999999 度，然后转上几年 100000000000 是时间
    moonContainer.rotateTo(new RotationData(0, 9999999999999, 0), 100000000000, MotionCurve.Linear);
}

与HTML不同HTML有很多节点而EMR只有三个节点分别是SpaceNode（空间节点)、PanelRoot（平面节点）、PanelLayer（平面图层节点）不断组合和复用这些节点就可以完成对整个空间的布局，再通过隐含的空间文档模型（Space Doucment Model) 就可以操纵和修改空间中的任意内容。
