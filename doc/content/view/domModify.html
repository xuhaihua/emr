文档操作

EMR中提供了一些对视图文档操作的基本方法，使用这些方法可以很方便的创建、修改和控制空间中的内容，它的作用类似于Web开发中对 Dom（文档对象模型）的操作，这些方法理解起来也非常容易

创建节点

下表列出的方法用于创建节点

方法 说明
protected PanelRoot createPanelRoot ( ... ) 创建PanelRoot节点
protected PanelLayer createPanelLayer ( ... ) 创建PanelLayer节点
protected SpaceNode createSpanceNode ( ... ) 创建SpaceNode节点
protected SpaceMagic createSpaceMagic () 创建SpaceMagic节点

下表列出的方法用于创建组件

查找元素

在系统内不管是节点还是组件它们都属于元素，元素与节点的区别是节点提供了操作的更多细节而元素只是一个接口，每个节点或组件在实现时都继承了它，元素抽取了所有节点和组件的公共属性。 下表列出了对元素的查询方法，元素的查找可以在文档、节点、组件下进行

方法 说明
protected List<EMR.Element> getElements () 获取视图文档内的所有元素
protected EMR.Element getElementById (string id) 在视图文档内按Id查找元素（如果指定的id不存在则返回null）
protected List<EMR.Element> getElementsByName (string name) 在当前组件的视图文档内按称名查找元素
方法 说明
List<EMR.Element> EMR.Entity.Node.getChildElements () 查找指定节点下的所有子元素
List<EMR.Element> EMR.Entity.Node.getChildElementsByName (string name) 查找指定节点下的所有名称为name的子元素
方法 说明
List<EMR.Element> EMR.Component.getChildElements () 查找指定组件下的所有子元素
List<EMR.Element> EMR.Component.getChildElementsByName (string name) 查找指定组件下的所有名称为name的子元素
