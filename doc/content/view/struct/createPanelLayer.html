createPanelLayer方法

创建平面图层节点

原型：

protected PanelLayer createPanelLayer (float? left, float? top, float? right, float? bottom, float width = 0, float height = 0, float depth = 0, int zIndex = 0, bool isPanel = true, NPC npc = null, RendMode rendMode = RendMode.opaque)

参数:

left： float? 节点left值
top： float? 节点top值
right： float? 节点right值
bottom： float? 节点bottom值
width： float 宽度，默认值为：0
height： float 高度，默认值为：0
depth： float 深度，默认值为：0
zIndex： int 层叠次序，默认值为：0
isPanel ： bool 是否为平面，默认值true
npc ： NPC NPC，默认值null
rendMode ： RendMode 渲染模式，默认值：RendMode.opaque
