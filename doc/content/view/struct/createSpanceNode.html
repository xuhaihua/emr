createSpanceNode方法

创建空间节点

原型：

protected PanelRoot createSpanceNode (float x, float y, float z, float xAngle, float yAngle, float zAngle, float width, float height, float depth, NPC npc = null, RendMode rendMode = RendMode.opaque)

参数:

x： float PanelRoot X轴坐标
y： float PanelRoot Y轴坐标
z： float PanelRoot Z轴坐标
width： float 宽度
height： float 高度
depth： float 深度
npc ： NPC NPC，默认值null
rendMode ： RendMode 渲染模式，默认值：RendMode.opaque
