verticalTopInterval

verticalTopInterval属性用于设置子节点垂直对齐时：起始节点顶部是否带有对齐间隔

  • verticalTopInterval = true

<>
    <!--
        创建一个平面容器，设置contentVertical属性为top控制其中的子节点靠顶部对齐
        设置verticalInterval属性为20控制各节点之间的间隔为20mm
        设置verticalTopInterval属性为true使开始节点顶部存在对齐间隔
     -->
    <>

        <!--子节点-->
        <></PanelLayer>
        <></PanelLayer>
        <></PanelLayer>
    </PanelRoot>
</Root>

  • verticalTopInterval = false

<>
    <!--
        创建一个平面容器，设置contentVertical属性为top控制其中的子节点靠顶部对齐
        设置verticalInterval属性为20控制各节点之间的间隔为20mm
        设置verticalTopInterval属性为false取消开始节点顶部的对齐间隔
     -->
    <>

        <!--子节点-->
        <></PanelLayer>
        <></PanelLayer>
        <></PanelLayer>
    </PanelRoot>
</Root>
