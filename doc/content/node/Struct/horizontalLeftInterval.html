horizontalLeftInterval

horizontalLeftInterval属性用于设置水平对齐时：起始节点左侧是否带有对齐间隔

  • horizontalLeftInterval = true

<>
    <!--
        创建一个平面容器，设置contentHorizontal属性为left控制其中的子节点靠左对齐
        设置horizontalInterval属性为20控制各节点之间的间隔为20mm
        设置horizontalLeftInterval属性为true使开始节点左侧存在对齐间隔
     -->
    <>

        <!--子节点-->
        <></PanelLayer>
        <></PanelLayer>
        <></PanelLayer>
    </PanelRoot>
</Root>

  • horizontalLeftInterval = false

<>
    <!--
        创建一个平面容器，设置contentHorizontal属性为left控制其中的子节点靠左对齐
        设置horizontalInterval属性为20控制各节点之间的间隔为20mm
        设置horizontalLeftInterval属性为false取消开始节点左侧的对齐间隔
     -->
    <>

        <!--子节点-->
        <></PanelLayer>
        <></PanelLayer>
        <></PanelLayer>
    </PanelRoot>
</Root>
