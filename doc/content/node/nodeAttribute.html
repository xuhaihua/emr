节点属性

本篇列出EMR中节点的一些常用属性，在视图文档中，如果一个属性的值引用的是一个变量，则该属性必须以 “_" 开头作为该属性的前缀（例如：如果 width 属性的值是变量 sizeWidth，那标签上的属性应该这样写 _width="sizeWidth"）

通用节点视图属性：

下表列出的属性在所有节点中都可以使用

名称 说明 数据类型 是否支持元素 是否样式 备注
id 节点的唯一标识 string 是 否 组件内唯一
name 节点名称 string 是 否 不同节点name属性可以相同
名称 说明 数据类型 是否支持元素 是否样式 备注
offset 偏移量 Vector3 是 是 默认值：0,0,0
width 宽度 float 是 是 单位 mm
height 高度 float 是 是 单位 mm
depth 厚度 或 深度 float 是 是 单位 mm
名称 说明 数据类型 是否支持元素 是否样式 备注
renderMode 节点渲染方式 可选的值有：opaque | cutout | fade | transparent | additive string enum 是 否 该属性只允许被设置一次
backgroundColor 节点背景色 Color String 是 是 Color类型的字符串表示（用“,”号分隔各颜色分量 ）例如：红色 255,0,0,0 默认为：无 "" 空字符串
backgroundImage 节点背景图 string 是 是 图片的Resources路径
borderWidth 边框大小，边框的强度等于灯光(lightIntensity)的强度该属性默认为0也就是不会被显示，边框的颜色等于悬浮灯光的颜色(hoverColor)默认为白色 float 是 是 单位(mm) 边框的设置与节点的lightIntensity、hoverColor属性有关，在设置边框时需要注意调节节点的这两个属性不然边框有可能不会被正常显示
borderRadius 节点圆角半径 float 是 是 单位 mm
stylesheet 与该节点关联的样式 string 是 否 和web前端中的元素的class属性是同一个概念，都是给节点赋于一个样式，只是在这里用的是stylesheet而不是class
名称 说明 数据类型 是否支持元素 是否样式 备注
horizontalFloat 节点在X轴上的浮动方式，可选的值有：none | center | left | right string enum 是 是 与web前端中的元素浮动是相同的概念 例如：在web中 float:left 表示将节点定位到父节点的最左侧 默认为：none
verticalFloat 节点在Y轴上的浮动方式，可选的值有： none | center | top | bottom string enum 是 是 默认为：none
contentHorizontal 子节点在X轴上的对齐方式，none | center | left | right | between string enum 是 是 默认为：none，这个功能类似于Web开发中的flex布局，作用轴为X
horizontalInterval 在X轴上各子节点之间的水平对齐间距 float 是 是 默认为：0
horizontalLeftInterval 在X轴上控制开始节点左侧是否存在对齐间隔 bool 是 是 默认为：false
horizontalRightInterval 在X轴上控制结束节点右侧是否存在对齐间隔 bool 是 是 默认为：false
contentVertical 子节点在Y轴上的对齐方式，可选的值有：none | center | top | bottom | between string enum 是 是 默认为：none，这个功能类似于Web开发中的flex布局，作用轴为Y
verticalInterval 在Y轴上各子节点之间的垂直对齐间距 float 是 是 默认为：0
verticalTopInterval 在Y轴上控制开始节点顶部是否存在对齐间隔 bool 是 是 默认为：false
verticalBottomInterval 在Y轴上控制结束节点底部是否存在对齐间隔 bool 是 是 默认为：false
名称 说明 数据类型 是否支持元素 是否样式 备注
lightIntensity 节点灯光强度 float 是 是 默认为：0
hoverColor 悬浮灯光颜色 Color String 是 是 Color类型的字符串表示（用“,”号分隔各颜色分量 ）例如：红色 255,0,0,0
名称 说明 数据类型 是否支持元素 是否样式 备注
widthFixed 节点宽度是否固定，限制宽度的变化 bool 是 是 PanelLayer节点默认值为true、PanelRoot节点默认值为：false、SpaceNode节点默认值为：false
heightFixed 节点高度是否固定，限制高度的变化 bool 是 是 PanelLayer节点默认值为true、PanelRoot节点默认值为：false、SpaceNode节点默认值为：false
depthFixed 节点深度或厚度是否固定，限制深度的变化 bool 是 是 PanelLayer节点默认值为true、PanelRoot节点默认值为：true、SpaceNode节点默认值为：false
enableManipulator 是否允许节点可操纵 bool 是 否 本能交互，使得用户可以通过手势或手部射移或旋转物体
名称 说明 数据类型 是否支持元素 是否样式 备注
npcPath NPC Resources路径 (加载一个NPC) string 是 是 NPC在Resources目录中的路径
npcOffset NPC 偏移量 Vector3 是 是 默认值：0,0,0
名称 说明 数据类型 是否支持元素 是否样式 备注
parasitifer 节点表演者 GameObject 否 否 这是节点的表现对象，所有属性将最终都作用于该对象上，所以你可以在代码中获取该对象，然后直接修改该对象上的属性或添加新的Unity Compontent

平面节点视图通用属性：

下表列出的属性括用于所有平面节点（PanelRoot节点、Element节点）

名称 说明 数据类型 是否支持元素 是否样式 备注
overflow 节点内容溢出显示方式，可选的值有：hidden | visible | scroll string enum 否 是 默认为：visible
名称 说明 数据类型 是否支持元素 是否样式 备注
color 文本颜色 Color String 否 是 Color类型的字符串表示（用“,”号分隔各颜色分量 ） 默认为白色： 255,255,255
fontSize 文本大小 float 否 是 单位(mm) 默认14
textHorizontal 文本X轴对齐方式，可选的值有：left | center | flush | geometry | justified | right string enum 否 是 默认为：left
textVertical 文本Y轴对齐方式，可选的值有：top | baseline | capline | geometry | middle | bottom string enum 否 是 默认为：top
characterSpace 字符间距 float 否 是 单位 mm
lineSpace 行高 float 否 是 单位 mm
名称 说明 数据类型 是否支持元素 是否样式 备注
scrollDirection 滚动方向，可选的值有：upAndDown | leftAndRight | both string enum 否 是 默认为：upAndDown

PanelRoot节点视图属性：

属于平面节点的一种，是空间内其他平面类型节点的根节点，可以把它理解为空间中的一个屏幕

名称 说明 数据类型 是否支持元素 是否样式 备注
x x轴坐标 float 否 是 默认为：0
y y轴坐标 float 否 是 默认为：0
z z轴坐标 float 否 是 默认为：0
xAngle x轴旋转量 float 否 是 默认为：0
yAngle y轴旋转量 float 否 是 默认为：0
zAngle z轴旋转量 float 否 是 默认为：0
名称 说明 数据类型 是否支持元素 是否样式 备注
forwardFloat 节点在Z轴上的浮动方式，可选的值有：none | center | back | forward string enum 否 是 默认为：none
名称 说明 数据类型 是否支持元素 是否样式 备注
xFixed x轴坐标是否固定，限制x轴的移动 bool 否 是 默认为：flase
yFixed y轴坐标是否固定，限制y轴的移动 bool 否 是 默认为：flase
zFixed z轴坐标是否固定，限制z轴的移动 bool 否 是 默认为：flase
xAngleFixed 限制节点沿x轴旋转 bool 否 是 默认为：flase
yAngleFixed 限制节点沿y轴旋转 bool 否 是 默认为：flase
zAngleFixed 限制节点沿z轴旋转 bool 否 是 默认为：flase
enableSize 节点是否启用手部尺寸本能交互 bool 否 否 默认为：flase
isPanel 是否为平面，只有当该属性为false时平面的depth（深度）属性才有效，该属性一旦被设置为false后后期将无法再更改 bool 否 否 默认为：true

PanelLayer节点视图属性：

属于平面节点的一种，该类节点的父节点只能是PanelRoot或PanelLayer，可以把它理解为html中的Div元素是用来构建布局的基本元素

名称 说明 数据类型 是否支持元素 是否样式 备注
zIndex 节点层叠次序 (你可以把它理解成web开发中div上的z-index属性，它们是同一个概念) int 否 是 默认为：0
left 当前节点的左侧与父节左侧的距离值 float 否 是 默认为：null
top 当前节点的顶部与父节点顶部的距离值 float 否 是 默认为：null
right 当前节点的右侧与父节点右侧的距离值 float 否 是 默认为：null
bottom 当前节点的底部与父节点底部的距离值 float 否 是 默认为：null
名称 说明 数据类型 是否支持元素 是否样式 备注
leftFixed 固定当前节点的左侧与父节左侧的距离值，限制节点的left属性值的变化 bool 否 是 默认为：true
topFixed 固定当前节点的顶部与父节点顶部的距离值，限制节点的top属性值的变化 bool 否 是 默认为：true

SpaceNode节点视图属性：

SpaceNode属于空间节点，以下列出的属性均可在该类节点上使用

名称 说明 数据类型 是否支持元素 是否样式 备注
x x轴坐标 float 否 是 默认为：0
y y轴坐标 float 否 是 默认为：0
z z轴坐标 float 否 是 默认为：0
xAngle x轴旋转量 float 否 是 默认为：0
yAngle y轴旋转量 float 否 是 默认为：0
zAngle z轴旋转量 float 否 是 默认为：0
名称 说明 数据类型 是否支持元素 是否样式 备注
forwardFloat 节点在Z轴上的浮动方式，可选的值有：none | center | back | forward string enum 否 是 默认为：none
bendAngle 空间弯曲角度 float 否 是 默认为：0
contentForward 子节点在Z轴上的对齐方式，可选的值有：none | center | back | forward | between string enum 否 是 默认为：none，这个功能类似于Web开发中的flex布局，作用轴为Z轴
forwardInterval 在Z轴上各子节点之间的对齐间距 float 否 是 默认为：0
forwardFrontInterval 结束节点正前侧也存在对齐间隔 bool 否 是 默认为：false
forwardBackInterval 开始节点正后侧是否存在对齐间隔 bool 否 是 默认为：false
名称 说明 数据类型 是否支持元素 是否样式 备注
overclip 溢出是否裁切（当他的子节点超出它的尺寸范围时是否裁切)，注意：该功能有一限制，如果要启用溢出裁切则他的所有父节点的overclip必须都要为false包括它的子节点，也就是在节点的同一深度方向只允许有一个节点的overclip为true bool 否 是 默认为：false
名称 说明 数据类型 是否支持元素 是否样式 备注
isRigidbody 是否为刚体，只有本属性的值为true时，节点的mass、drag、angularDrag、useGravity、isKinematic属性才会有效 bool 否 否 默认值：false
mass 质量 float 否 否 默认值：1
drag 阻力 float 否 否 默认值：0
angularDrag 角阻力 float 否 否 默认值：0.05f
useGravity 是否使用重力 bool 否 否 默认值：true
isKinematic 是否使用逆向运动学 bool 否 否 默认值：false
名称 说明 数据类型 是否支持元素 是否样式 备注
xFixed x轴坐标是否固定，限制节点在x轴上移动 bool 否 是 默认为：flase
yFixed y轴坐标是否固定，限制节点在y轴上移动 bool 否 是 默认为：flase
zFixed z轴坐标是否固定，限制节点在z轴上移动 bool 否 是 默认为：flase
xAngleFixed 限制节点沿x轴旋转 bool 否 是 默认为：flase
yAngleFixed 限制节点沿y轴旋转 bool 否 是 默认为：flase
zAngleFixed 限制节点沿z轴旋转 bool 否 是 默认为：flase
joint 节点关节（scale坐标），默认值就为：null Vector3? 否 否 默认为：null
jointRotationAxle 关节旋转轴 Vector3? 否 否 默认为：null
jointConnectId 关节连接对象的id string 否 否 默认为：空
enableSize 节点是否启用手部尺寸本能交互 bool 否 否 默认为：flase
handleSize 尺寸修改组件控制柄的大小 float 否 否 默认为：0.016，注意：该值的单位并不是(mm)，它的值其实是比例
sizeBoxPadding 尺寸修改组件控制柄与节点之间的间距 float 否 否 单位(mm) ，默认为：0
